# Re:um

산업통상자원부의 폐합성수지 산업공생 정책 운영을 지원하는 통합 대시보드입니다.

## 주요 기능

- 폐합성수지 배출·중간처리·수요기업 컨소시엄 추천
- 기업 참여 및 승인 진행 현황 관리
- 지역·기간·수지 종류별 탄소감축·재활용 성과 분석
- 정책별 참여기업, 예산 집행률 및 사업 진행 현황 확인
- 폐합성수지 환경성과 리포트 생성
- 자원순환 정책 운영 성과 리포트 생성
- 지원사업 추천 및 정책 알림 제공

## 로컬 실행

```powershell
npm.cmd install
npm.cmd run dev
```

터미널에 표시되는 로컬 주소로 접속합니다.
